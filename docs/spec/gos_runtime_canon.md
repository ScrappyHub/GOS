# GOS Runtime Canon

## What GOS is
A deterministic host.

## What GOS is not
Not an umbrella.
Not an orchestrator.
Not a database.
Not a cloud.

## Responsibilities

MUST:
- verify bundles
- enforce policies
- run workloads
- seal snapshots
- emit commitments
- duplicate to NFL
- maintain NeverLost identity

MUST NOT:
- modify artifacts silently
- skip signing
- skip receipts
- call other projects directly
